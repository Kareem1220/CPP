# C++ Repository

A comprehensive, organized collection of C++ programs and projects — ranging from the absolute fundamentals of the language to advanced system-level programming, algorithms, and software design. Every section is structured for clarity and progressive learning.

---

## 📁 Contents

### 🔰 Basic C++
Core language fundamentals including syntax, data types, variables, operators, control flow, loops, functions, recursion, arrays, pointers, and references. The foundation everything else builds on.

### 🧩 Object-Oriented Programming (OOP)
Projects and exercises implementing the four pillars of OOP in C++: **Encapsulation**, **Abstraction**, **Inheritance**, and **Polymorphism**. Includes real-world class designs, constructors/destructors, operator overloading, and interface modeling.

### 📊 Data Structures & Algorithms (DSA)
From-scratch implementations of essential data structures — Arrays, Linked Lists, Stacks, Queues, Trees (BST, AVL, Segment), Graphs, Heaps, and Hash Tables — paired with algorithms covering Sorting, Searching, Dynamic Programming, Backtracking, Divide & Conquer, and Greedy approaches.

### 📦 Standard Template Library (STL)
Practical use of the C++ STL: `vector`, `list`, `deque`, `map`, `unordered_map`, `set`, `stack`, `queue`, `priority_queue`, `pair`, `tuple`, iterators, and algorithm utilities like `sort`, `find`, `transform`, and more.

### 🧵 Multi-threading & Concurrency
Programs exploring concurrent C++ using `std::thread`, `std::mutex`, `std::condition_variable`, `std::async`, `std::future`, and atomic operations — covering thread safety, deadlocks, and synchronization patterns.

### 🔧 Templates & Generic Programming
Function templates, class templates, variadic templates, template specialization, and SFINAE — writing flexible and reusable C++ code that works across types.

### 🎨 Design Patterns
C++ implementations of the classic Gang of Four (GoF) design patterns across all three categories — Creational (Singleton, Factory, Builder), Structural (Adapter, Decorator, Proxy), and Behavioral (Observer, Strategy, Command).

### 📂 File I/O & Serialization
Reading and writing files using `fstream`, parsing structured data, binary I/O, and serialization/deserialization of objects to and from disk.

### 🧠 Memory Management
Deep dives into manual memory management, smart pointers (`unique_ptr`, `shared_ptr`, `weak_ptr`), RAII, custom allocators, memory leak detection, and the difference between stack and heap.

### 🏆 Competitive Programming
Optimized C++ solutions to problems from platforms like **LeetCode**, **Codeforces**, and **HackerRank** — focused on time complexity, space complexity, and clean implementation under constraints.

### 🖥️ System Programming
Low-level C++ programs interfacing with the OS — covering process management, socket programming, signal handling, and working with system calls.

### 🎮 Mini Projects
Standalone C++ applications including console games, simulators, and utility tools — built to apply multiple concepts together in a practical, end-to-end context.

## 🛠️ How to Compile & Run

**Single file:**
```bash
g++ -std=c++17 -o output filename.cpp
./output
```

**With CMake:**
```bash
mkdir build && cd build
cmake ..
make
./your_program
```

- CMake *(optional — for multi-file projects)*
